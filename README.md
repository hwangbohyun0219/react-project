#### 프로젝트 소개
+ react.js, JavaScript와 heidSQL을 활용하여 로그인과 회원가입을 하고 직접 게시물에 댓글을 달 수 있는 sns사이트를 만들었습니다.
+ 카테고리를 선택하고 제목을 입력한 다음 직접 이미지도 추가하여 피드에 새로운 게시물을 업로드할 수 있습니다.
